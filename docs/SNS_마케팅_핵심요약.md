# 헬창마을(HCM) — 인스타/스레드 마케팅 핵심 비즈니스 요약

> 작성 기준: BE-HCM(Spring Boot) / FE-HCM(React PWA) 코드베이스 실측 분석
> 용도: SNS 카피라이팅, 콘텐츠 기획, 세일즈 피칭

---

## 1. 서비스 한 줄 소개

> **"운동 안 한 사람 찾아내고, 벌금까지 계산해주는 건 이제 앱이 합니다."**

**메인 카피 (사용자 관점)**
> 친구랑 "이번 주부터 진짜 운동하자" 하고 만든 그 단톡방,
> **누가 빠졌는지 세는 사람만 지쳐서 없어집니다.**
> 헬창마을은 그 계산을 매주 월요일 자정에 앱이 대신 합니다.

**서브 카피 배리에이션**
- 의지로 안 되니까, 시스템으로 합니다.
- 사진 한 장이면 인증 끝. 벌금 계산은 자동.
- 단톡방의 총무를 없애드립니다.

## 2. 서비스 분야 및 카테고리

| 구분 | 내용 |
|------|------|
| **1차 도메인** | 헬스케어 / 피트니스 — 운동 습관 형성(Habit Formation) |
| **2차 도메인** | 소셜 커뮤니티 SaaS — 그룹 챌린지·책임 관리(Group Accountability) |
| **비즈니스 모델 유형** | B2C 그룹 챌린지 플랫폼 (현재 **베타 · 전 기능 무료**, `PricingSection.tsx`) |
| **인접 카테고리** | 행동경제학 기반 커미트먼트 서비스(Commitment Device) |
| **제품 형태** | 설치형 PWA (앱스토어 심사 불필요, 홈 화면 추가 + 웹 푸시) |

### 핵심 경쟁력 기능 TOP 3

**① 벌금 자동 집계 엔진 (진짜 차별점)**
- 매주 **월요일 자정 크론 잡**이 자동 실행 (`WorkoutSchedulingService`, `0 0 0 * * MON`)
- 지난주 월~일 인증 횟수를 집계 → `(목표 횟수 − 실제 인증) × 1회당 벌금` 자동 산출
- 벌금 부과 즉시 당사자에게 푸시 발송: *"💸 벌금이 부과됐어요"*
- 집계 직후 주간 카운트 자동 리셋 → 사람이 개입할 구석이 없음

**② 원터치 운동 인증 + 다중 방 동시 반영**
- 사진(최대 3장, JPEG/PNG/WebP) + 종목 + 시간 입력으로 인증 완료
- **한 번 인증하면 참여 중인 모든 운동방에 동시 반영** (`WorkoutService.authenticateWorkout`)
- 같은 날 중복 인증은 서버가 차단 (`WORKOUT_ALREADY_AUTHENTICATED`) → 어뷰징 방지
- 인증 성공 시 **1440×1440 공유 카드 이미지를 자동 생성**해 네이티브 공유 시트 호출
  → **인스타 스토리·피드에 바로 올라가는 구조** (바이럴 루프가 제품에 내장되어 있음)

**③ 방 단위 실시간 채팅 + 3종 행동 유도 푸시**
- STOMP/WebSocket 실시간 채팅(이미지 전송, 읽음 표시, 무한 스크롤) → 카톡방 대체
- 푸시 3종이 각각 다른 심리 레버를 담당:

| 시점 | 메시지 | 심리 장치 |
|------|--------|-----------|
| 매일 21시 | "⏰ 오늘 운동 인증 잊지 마세요 — OO방에서 오늘 **3명이 인증했어요**!" | 사회적 증거 + 소외 불안 |
| 목표 달성 순간 | "🎉 OO님이 이번 주 목표(3회)를 달성했어요!" (방 전체 발송) | 인정 욕구 + 경쟁 |
| 월요일 자정 | "💸 벌금이 부과됐어요" | 손실 회피 |

## 3. 주요 타깃 고객 (페르소나 3)

### 페르소나 A — "작심삼일 루프에 갇힌 직장인" (핵심 사용자, 26~35세)

| 항목 | 내용 |
|------|------|
| **Pain point** | 헬스장 1년 등록해놓고 2주 만에 증발. "내일 하지 뭐"가 반복되는데 아무도 뭐라 안 함. 혼자 하면 기준도, 마감도, 대가도 없음. |
| **Trigger 상황** | 건강검진 결과, 결혼식/여행 D-day, 친구의 인증샷 |
| **우리의 해결책** | 주간 목표 미달 시 **실제 돈이 나가는 마감(월요일 자정)** 을 만들어 줌. 게다가 방 멤버 전원이 내 미인증 상태를 봄. |
| **핵심 가치** | "의지가 아니라 시스템". 매일 21시 리마인더가 하루도 그냥 지나가지 않게 만듦 |
| **소구 카피** | "돈 나가는 게 싫어서 나간 헬스장, 이번엔 진짜 갑니다." |

> 참고: 코드상 **1인용 운동방 개설 가능**(`maxMembers` 최소 1). "친구 모을 자신 없는 사람"도 온보딩 가능 → 유입 장벽 낮추는 소구점으로 활용.

### 페르소나 B — "총무병에 걸린 모임장 / 방장" (구매 결정권자, 유입 허브)

| 항목 | 내용 |
|------|------|
| **Pain point** | 러닝크루·헬스 소모임 운영 중. 매주 단톡방 스크롤 올려가며 누가 몇 번 했는지 손으로 세고, 엑셀에 옮기고, 벌금 걷다가 사이 어색해짐. "야 너 이번 주 두 번밖에 안 했잖아"를 **내 입으로** 말해야 함. |
| **Trigger 상황** | 3주차쯤 집계가 밀리고, 모임이 조용히 붕괴되는 시점 |
| **우리의 해결책** | 규칙(주 N회 / 1회당 벌금 / 정원)을 방 개설 시 한 번만 설정 → 이후 **집행은 서버가 대신**. 벌금 계좌(은행·계좌번호·예금주)를 방에 등록해두면 정산 안내도 자동. |
| **핵심 가치** | 악역 대행. 관계를 소모하지 않고 규칙을 유지 |
| **소구 카피** | "누가 빠졌는지 말하는 건, 이제 방장이 아니라 앱이 합니다." |

**설정 가능 범위 (코드 실측 · `workoutRoomRules.ts`)**
- 주간 최소 운동 횟수: **1~7회**
- 1회 누락당 벌금: **1,000~50,000원**
- 정원: **1~10명**
- 입장 코드 기반 초대 (**카카오톡 공유** 지원, 미설정 시 링크 복사로 자동 대체)

### 페르소나 C — "소규모 그룹을 관리하는 트레이너·코치·챌린지 운영자"

| 항목 | 내용 |
|------|------|
| **Pain point** | PT 수업 있는 날만 운동하고 나머지 5일은 증발하는 회원들. 수업 밖 행동을 확인할 방법이 없어 결과가 안 나오고, 결과가 안 나오니 재등록이 끊김. |
| **Trigger 상황** | 그룹 챌린지·바디프로필 프로그램 운영, 사내 웰니스 챌린지 기획 |
| **우리의 해결책** | 클라이언트를 방 단위(최대 10명)로 묶어 **수업 외 요일의 인증까지 데이터로 확보**. 관리자 계정은 여러 방 동시 참여·관리 가능(`/admin/**`, `ROLE_ADMIN`)하며, 회원·운동방 목록/상세 관리 화면 제공. |
| **핵심 가치** | 결과를 만들어내는 도구 = 재등록률 방어 수단 |
| **소구 카피** | "수업 없는 날에 결과가 갈립니다. 그 5일을 관리하세요." |

## 4. 마케팅 소구점 (USP)

### USP ①. "규칙을 사람이 아니라 서버가 집행한다"

대체재(단톡방 + 엑셀 + 총무의 잔소리)와의 결정적 차이는 **집행의 자동화와 공정성 설계**입니다. 단순히 "기록해주는 앱"이 아니라, **판정 로직 자체가 서버에 박혀 있어 감정이 개입할 수 없는 구조**입니다.

코드에서 확인되는 공정성 장치:
- **자동 집계**: 매주 월요일 자정, 지난주 월~일 기준으로 일괄 산정 (사람의 재량 없음)
- **휴식일 예외**: 부상·여행·시험 기간을 사전 등록하면 그 주는 벌금 면제 (`Rest` 기간이 해당 주와 겹치면 스킵)
- **소급 적용 차단**: 벌금 제도를 켜고 끄거나 금액을 바꿔도 **최소 7일 뒤 첫 월요일부터** 적용 (`getEarliestPenaltyEffectiveMonday` / `applyDuePendingPenaltyChangeIfDue`)
  → 방장이 벌금 규칙을 갑자기 올려서 멤버를 털 수 없음. **"룰이 뒤에서 바뀌지 않는다"** 는 신뢰가 곧 리텐션.
- **중복 인증 차단**: 같은 날 두 번 인증 불가, 과거 날짜 인증은 7일 이내로 제한

> **카피 방향**: "봐주는 사람도, 삐지는 사람도 없습니다. 월요일 자정에 서버가 계산합니다."

---

### USP ②. "돈의 압박은 설계하되, 돈은 받지 않는다"

대부분의 챌린지 서비스는 **예치금을 먼저 걸게** 합니다. 그 순간 가입 전환율이 무너집니다. 헬창마을은 **결제·송금 기능을 아예 두지 않고, 계산과 기록만 투명하게 제공**합니다(FAQ에 명시). 실제 정산은 등록된 벌금 계좌로 멤버끼리 직접.

이 설계가 만드는 마케팅적 우위:
- **가입 마찰 0** — 카드 등록·예치금 없음. 구글/카카오/네이버 소셜 로그인으로 "1초 가입" 소구 가능
- **설치 마찰 0** — PWA라 앱스토어를 거치지 않고 홈 화면 추가 + 푸시 알림 수신. 인스타 프로필 링크 → 즉시 체험까지 이탈 지점이 없음
- **비용 마찰 0** — 베타 기간 전 기능 무료
- **심리적 마찰 0** — 내 돈이 플랫폼에 묶이지 않으므로 "일단 해볼까"의 문턱이 낮음

> **카피 방향**: "예치금 없습니다. 카드 등록도 없습니다. 벌금은 친구한테 내세요. 우리는 계산만 해드립니다."

## 5. SNS 콘텐츠 실행 가이드 (인스타 / 스레드)

### 5-1. 훅(Hook) 카피 — 스레드용 첫 문장

1. 3주 만에 사라진 운동 단톡방, 원인은 의지가 아니라 **총무**였습니다.
2. "이번 주 두 번밖에 안 했잖아"를 말해야 하는 사람이 제일 먼저 지칩니다.
3. 운동 안 하면 5천 원. 근데 그 5천 원을 **아무도 안 걷어요.** 그게 진짜 문제.
4. 헬스장 등록비보다 비싼 건, 안 가면서 내는 12개월 할부입니다.
5. 매주 월요일 자정, 우리 방에서 벌금 폭탄 맞는 사람이 정해집니다.
6. 운동 인증 앱 만들면서 제일 신경 쓴 건 운동이 아니라 **인간관계**였습니다.

### 5-2. 콘텐츠 필러 (4주 로테이션)

| 필러 | 포맷 | 소재 |
|------|------|------|
| **문제 공감** | 스레드 텍스트 | 작심삼일·총무 번아웃·단톡방 소멸기 |
| **기능 데모** | 릴스/GIF | 인증 10초 컷 → 대시보드 자동 집계 → 월요일 벌금 알림 (실제 `dashboard.gif`, `upload.gif`, `penalty.gif` 자산 보유) |
| **UGC 확산** | 인증 카드 리그램 | 인증 완료 시 자동 생성되는 1440×1440 공유 카드 (`#운동인증 #HCM` 해시태그 기본 삽입) |
| **소셜 프루프** | 카드뉴스 | 랜딩의 실시간 지표 API(총 인증 횟수 / 운영 중인 방 수 / 누적 유저) 스크린샷 |

### 5-3. 전환 동선

```
인스타 프로필 링크 → 랜딩(bellwin.co.kr) → 소셜 로그인(1초)
→ 방 개설 or 입장 코드 참여 → 카카오톡으로 친구 초대 → 첫 인증
→ 자동 생성된 인증 카드를 인스타 공유 → 신규 유입 (바이럴 루프)
```

**핵심 KPI 제안**: 방 개설 전환율 / 방당 평균 초대 수락 수 / 첫 주 인증 3회 달성률 / 인증 카드 공유율

## 6. 마케팅 시 주의사항 (과장 금지 라인)

코드 기준으로 **아직 말하면 안 되는 것들**입니다.

| 항목 | 실제 상태 | 대응 |
|------|-----------|------|
| **결제·송금** | 미지원 (의도적 설계) | "정산 자동화"❌ → "**계산 자동화**"⭕ 로 표현 |
| **벌금 납부 이력 조회** | 프론트 `getPenaltyPayments`가 아직 목(mock) 데이터 | 납부 완료 처리·영수증 관련 문구는 사용 금지 |
| **AI 사진 검증** | 미지원 (FAQ에 "사진 검증하지 않음" 명시) | "AI가 판별"❌ → "**멤버끼리 서로 확인**"⭕ |
| **누적 지표** | 현 규모 초기 단계 (랜딩 폴백값 기준 수십 명대) | 절대 수치보다 **참여율·달성률** 서사로 소구 |
| **주식 포트폴리오 기능** | FAMILY/ADMIN 전용 내부 기능 | 대외 마케팅 메시지에서 제외 (포지셔닝 흐림) |

## 7. 30초 엘리베이터 피치

> 헬창마을은 **벌금 관리형 운동 인증 플랫폼**입니다.
> 친구들과 운동방을 만들고 "주 3회, 못 하면 5천 원" 같은 규칙만 정하면,
> 인증은 사진 한 장으로 10초면 끝나고 **매주 월요일 자정에 서버가 자동으로 벌금을 집계**합니다.
> 휴식일 예외도, 규칙 변경 소급 방지도 앱 안에 들어 있어서 **총무 없이도 관계가 상하지 않습니다.**
> 예치금도 카드 등록도 없고, 설치 없이 웹에서 바로 시작하며, 베타 기간 전 기능 무료입니다.
> **의지가 아니라 시스템으로 운동 습관을 만드는 서비스**입니다.
